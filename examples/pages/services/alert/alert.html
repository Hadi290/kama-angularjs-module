// simple alert
alertService.success('متن تست');

// unique alert
alertService.info('متن تست', { unique: true });

// custom timeout
alertService.warning('متن تست', { timeout: 10000 });

alertService.actionName(message, options);
name arguments notes
success, info, warning, error message=[string], options=[object]
name type default value notes
message string null sould be a string to display as an alert. if it's a falsy value alert won't show.
options object {} you can pass an optional object as options.
name type default value notes
unique boolean false if unique set to be true, multiple call of that alert display only one alert.
timeout number 5000 time that alert shows before it disappears, in milliseconds.
if set to -1 alert never disappears.
